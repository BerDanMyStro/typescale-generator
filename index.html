Type Scale Generator

Create beautiful, responsive typography with ease

Settings

Uses clamp() function for responsive scaling

Font sizes will scale smoothly between viewport sizes

Preview

SCSS Variables

// Type Scale Variables
$base-font-size: 16px;
$scale-factor: 1.2;

// Type Scale Map
$type-scale: (
  'xs': 0.75rem,    // 12px
  'sm': 0.875rem,   // 14px
  'base': 1rem,     // 16px
  'lg': 1.125rem,   // 18px
  'xl': 1.25rem,    // 20px
  '2xl': 1.5rem,    // 24px
  '3xl': 1.875rem,  // 30px
  '4xl': 2.25rem,   // 36px
  '5xl': 3rem,      // 48px
  '6xl': 3.75rem,   // 60px
);

// Usage examples:
// font-size: map-get($type-scale, 'lg');
// or with a mixin:
@mixin font-scale($size) {
  font-size: map-get($type-scale, $size);
}
